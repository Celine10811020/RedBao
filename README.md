# 經文紅包｜Bible Blessing Red Envelope

A festive web-based red envelope designed for sharing Bible verses and blessings with friends during Chinese New Year.

Built with **HTML, CSS, and JavaScript**, this project combines scripture, animation, sound effects, and seasonal visual design into a lightweight interactive webpage.

## Live Demo

[Open Red Envelope](https://celine10811020.github.io/RedBao/)

## About the Project

This project is a digital blessing red envelope website.

Instead of containing money, this red envelope delivers a meaningful Bible verse and blessing message.  
It is designed as a festive webpage that can be shared with friends during Chinese New Year.

## Features

- Digital red envelope style interface
- Bible-based blessing messages
- Random verse content from a scripture collection
- Festive animation effects
- Background sound support
- Easy sharing through a static webpage
